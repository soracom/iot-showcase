# AWS IoT 1-Click で自動生成された Lambda 関数や IAM ロール/ポリシー

これらは不要であれば全て削除可能です。

* AWS Lambda 関数
    * `iot1click_onclick_*`
* AWS IAM
    * ロール
        * `iot1click_onclick_*`
    * ポリシー
        * `AWSLambdaIoTButtonSESExecutionRole-*` (Email の場合)
        * `AWSLambdaIoTButtonSNSExecutionRole-*` (SMS の場合)
        * `AWSLambdaIoTButtonBasicExecutionRole-*`
